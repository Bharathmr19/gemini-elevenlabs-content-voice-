About VoicePromptAI

VoicePromptAI is a web application that generates articles using the Gemini API and reads them aloud using the ElevenLabs Text-to-Speech API.

This application demonstrates the power of combining AI text generation with advanced text-to-speech technology.

Features:

  • Generate high-quality content with Gemini AI
  • Convert text to lifelike speech with ElevenLabs
  • Multiple voice options
  • Download audio files
